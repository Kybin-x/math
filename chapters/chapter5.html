第五章 · 数列

找规律、算总和——数列就是"排好队的数字"🧮

等差数列 等比数列
📊 知识点一：等差数列
🪜 爬楼梯，每层都高3米：$0, 3, 6, 9, 12, ...$，每相邻两个数的差都是 $3$，这就是等差数列。那个固定的差叫公差 $d$。
📐 等差数列的通项公式 $$a_n = a_1 + (n-1)d$$ $$\text{其中 } a_1 \text{ 是首项，} d \text{ 是公差}$$
$a_1$：第一个数（起点）
$d$：每次增加多少（步长）
$n$：第几个数

💡 理解方式：从第1项出发，走 $(n-1)$ 步，每步走 $d$，就到了第 $n$ 项。
📐 等差数列的前n项和 $$S_n = \frac{n(a_1 + a_n)}{2} = \frac{n[2a_1 + (n-1)d]}{2}$$
第一个公式更好记：前 $n$ 项和 = 项数 × (首项 + 末项)÷ 2。
💡 就像高斯算 $1+2+...+100$ 的方法：首尾配对！
$(1+100) \times 100 \div 2 = 5050$
📐 等差中项 $$\text{若 } a, A, b \text{ 成等差数列，则 } A = \frac{a+b}{2}$$
✏️ 例题1 入门

等差数列 $2, 5, 8, 11, ...$，求第 $10$ 项。

$a_1 = 2$，$d = 5- 2 = 3$
$a_{10} = a_1 + (10-1)d = 2 + 9 \times 3 = 2 + 27 = 29$
✅ 答案：$a_{10} = 29$
✏️ 例题2 入门

求等差数列 $1, 3, 5, 7, ...$ 的前 $20$ 项和。

$a_1 = 1$，$d = 2$
$a_{20} = 1 + 19 \times 2 = 39$
$S_{20} = \dfrac{20 \times (1 + 39)}{2} = \dfrac{20 \times 40}{2} = 400$
✅ 答案：$S_{20} = 400$
✏️ 例题3 进阶

等差数列中，$a_3 = 7$，$a_7 = 19$，求 $a_1$ 和 $d$。

$a_3 = a_1 + 2d = 7$ ...①
$a_7 = a_1 + 6d = 19$ ... ②
② - ①：$4d = 12$，$d = 3$
代入①：$a_1 + 6 = 7$，$a_1 = 1$
✅ 答案：$a_1 = 1$，$d = 3$
✏️ 例题4 挑战

等差数列 $\{a_n\}$，$S_n$ 为前 $n$ 项和，已知 $S_{10} = 100$，求 $a_5+ a_6$。

$S_{10} = \dfrac{10(a_1+a_{10})}{2} =5(a_1+a_{10}) = 100$
$a_1 + a_{10} = 20$
等差数列性质：$a_1+a_{10} = a_5+a_6$（首尾对称）
✅ 答案：$a_5 + a_6 = 20$
📈 知识点二：等比数列
🦠 细菌分裂：$1, 2, 4, 8, 16, ...$，每个数都是前一个数的$2$ 倍，这就是等比数列。那个固定的倍数叫公比 $q$。
📐 等比数列的通项公式 $$a_n = a_1 \cdot q^{n-1}$$ $$\text{其中 } a_1 \text{ 是首项，} q \text{ 是公比}(q \neq 0)$$
$a_1$：第一个数
$q$：每次乘以多少（倍率）

💡 理解方式：从第1项出发，乘 $(n-1)$ 次$q$，就到了第 $n$ 项。
📐 等比数列的前n项和 $$S_n = \begin{cases} na_1, & q = 1 \\ \dfrac{a_1(1-q^n)}{1-q}, & q \neq 1 \end{cases}$$
当 $q = 1$ 时，数列是常数列，如 $3,3,3,...$，前 $n$ 项和就是 $3n$。
当 $q \neq 1$ 时用公式。

💡 另一种写法：$S_n = \dfrac{a_1 - a_n \cdot q}{1 - q}$，有时更方便。
📐 等比中项 $$\text{若 } a, G, b \text{ 成等比数列，则 } G^2 = ab$$
✏️ 例题1 入门

等比数列 $1, 2, 4, 8, ...$，求第 $8$ 项。

$a_1 = 1$，$q = 2$
$a_8 = 1 \times 2^{8-1} = 2^7 = 128$
✅ 答案：$a_8 = 128$
✏️ 例题2 入门

等比数列 $3, 6, 12, 24, ...$，求前 $5$ 项和。

$a_1 = 3$，$q = 2$，$q \neq 1$
$S_5 = \dfrac{3(1-2^5)}{1-2} = \dfrac{3(1-32)}{-1} = \dfrac{3 \times (-31)}{-1} = 93$
验证：$3+6+12+24+48 = 93$ ✓
✅ 答案：$S_5 = 93$
✏️ 例题3 进阶

等比数列中，$a_2 = 6$，$a_5 = 162$，求 $a_1$ 和 $q$。

$a_2 = a_1 q = 6$ ... ①
$a_5 = a_1 q^4 = 162$ ... ②
② ÷ ①：$q^3 = \dfrac{162}{6} = 27$
$q = 3$
代入①：$a_1 \times 3 = 6$，$a_1 = 2$
✅ 答案：$a_1 = 2$，$q = 3$
✏️ 例题4 挑战

等比数列 $\{a_n\}$，$a_1 = 1$，$q = -2$，求 $S_6$。

$q = -2 \neq 1$，用公式
$S_6 = \dfrac{1 \times (1-(-2)^6)}{1-(-2)} = \dfrac{1-64}{1+2} = \dfrac{-63}{3} = -21$
验证：$1+(-2)+4+(-8)+16+(-32) = 1-2+4-8+16-32 = -21$ ✓
✅ 答案：$S_6 = -21$
🧪 第五章 · 小测验
1. 等差数列 $3, 7, 11, 15, ...$ 的公差是？
💡 $d = 7 - 3 = 4$。
2. 等差数列 $a_1 = 5$，$d = 3$，则$a_{20}$ 等于？
💡 $a_{20} = 5 + 19 \times 3 = 5 + 57 = 62$。
3. 求$1+2+3+...+100$ 的值？
💡 $S_{100} = \dfrac{100 \times (1+100)}{2} = 50 \times 101 = 5050$。高斯的经典！
4. 等比数列 $2, 6, 18, 54, ...$ 的公比是？
💡 $q = \dfrac{6}{2} = 3$。
5. 等比数列 $a_1 = 3$，$q = 2$，则 $a_6$ 等于？
💡 $a_6 = 3 \times 2^5 = 3 \times 32 = 96$。
6. 等差数列中，$a_1 + a_9 = 20$，则 $a_5$ 等于？
💡 等差数列性质：$a_1+a_9 = 2a_5$（首尾对称）。$2a_5=20$，$a_5=10$。
7. 等比数列 $1, -2, 4, -8, ...$ 的公比是？
💡 $q = \dfrac{-2}{1} = -2$。公比可以是负数！
8. 等差数列 $2, 5, 8, 11, ...$ 的前 $10$ 项和是？
💡 $a_{10}=2+9\times3=29$，$S_{10}=\dfrac{10(2+29)}{2}=\dfrac{310}{2}=155$。
9. 若$a$，$6$，$b$ 成等比数列，则 $ab$ 等于？
💡 等比中项：$6^2 = ab$，$ab = 36$。
10. 等比数列 $a_1=1$，$q=3$，则 $S_4$ 等于？
💡 $S_4 = \dfrac{1(1-3^4)}{1-3} = \dfrac{1-81}{-2} = \dfrac{-80}{-2} = 40$。验证：$1+3+9+27=40$ ✓
