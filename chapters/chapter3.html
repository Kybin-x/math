第三章 · 函数

输入一个数，输出一个结果——函数就是数学里的"自动贩卖机" 🎰

函数的概念 一次与二次函数 指数与对数函数
🔧 知识点一：函数的概念
🎰 你往自动贩卖机里投一枚硬币（输入 $x$），它就吐出一瓶饮料（输出 $y$）。每投一枚硬币，只会出一瓶饮料，不会出两瓶。这就是函数：每个输入对应唯一一个输出。
📐 函数的定义 $$y = f(x)$$ $$\text{定义域：x 的取值范围（能投什么硬币）}$$ $$\text{值域：y 的取值范围（能出什么饮料）}$$
函数三要素：定义域、对应法则、值域。
两个函数相同，当且仅当定义域和对应法则都相同。

常见求定义域的限制：
• 分母不能为零：$\dfrac{1}{x}$ → $x \neq 0$
• 偶次根号下不能为负：$\sqrt{x}$ → $x \geq 0$
• 对数的真数必须为正：$\ln x$ → $x > 0$
✏️ 例题1 入门

求函数 $f(x) = \dfrac{1}{x-2}$ 的定义域。

分母不能为零：$x - 2 \neq 0$
$x \neq 2$
✅ 答案：$\{x \mid x \neq 2\}$，即 $(-\infty, 2) \cup (2, +\infty)$
✏️ 例题2 入门

求函数 $f(x) = \sqrt{3-x}$ 的定义域。

根号下不能为负：$3 - x \geq 0$
$x \leq 3$
✅ 答案：$(-\infty, 3]$
✏️ 例题3 进阶

求函数 $f(x) = \dfrac{\sqrt{x-1}}{x-3}$ 的定义域。

条件1：根号下 $\geq 0$：$x - 1 \geq 0 \Rightarrow x \geq 1$
条件2：分母 $\neq 0$：$x - 3 \neq 0 \Rightarrow x \neq 3$
取交集：$x \geq 1$ 且 $x \neq 3$
✅ 答案：$[1, 3) \cup (3, +\infty)$
✏️ 例题4 入门

已知 $f(x) = 2x^2 - 3x + 1$，求 $f(2)$ 和 $f(-1)$。

$f(2) = 2 \times 4 - 3 \times 2 + 1 = 8 - 6 + 1 = 3$
$f(-1) = 2 \times 1 - 3 \times (-1) + 1 = 2 + 3 + 1 = 6$
✅ 答案：$f(2) = 3$，$f(-1) = 6$
📉 知识点二：一次函数与二次函数
📏 一次函数画出来是一条直线，二次函数画出来是一条抛物线。它们是高考中最常考的两类函数，掌握好图像和性质就能拿分！
📐 一次函数 $$y = kx + b \quad (k \neq 0)$$ $$k > 0 \text{：从左到右上升（递增）}$$ $$k < 0 \text{：从左到右下降（递减）}$$
$k$ 是斜率（决定倾斜程度），$b$ 是截距（直线与 y 轴的交点）。
$k > 0$：越往右越高；$k < 0$：越往右越低。
📐 二次函数 $$y = ax^2 + bx + c \quad (a \neq 0)$$ $$\text{顶点坐标：}\left(-\frac{b}{2a}, \; \frac{4ac-b^2}{4a}\right)$$ $$\text{对称轴：} x = -\frac{b}{2a}$$
$a > 0$：开口朝上（碗形 🥣），顶点是最低点
$a < 0$：开口朝下（帽形 🎩），顶点是最高点

💡 顶点公式记忆技巧：
x 坐标：$-\dfrac{b}{2a}$（"负b除以2a"）
y 坐标：把 x 坐标代入原函数算出来就行
✏️ 例题1 入门

函数 $y = -2x + 3$ 是递增还是递减？与 y 轴交于哪一点？

$k = -2 < 0$，所以是递减函数
$b = 3$，与 y 轴交于 $(0, 3)$
✅ 答案：递减，交于 $(0, 3)$
✏️ 例题2 进阶

求二次函数 $y = x^2 - 4x + 3$ 的顶点坐标和对称轴。

$a=1, b=-4, c=3$
对称轴：$x = -\dfrac{-4}{2 \times 1} = 2$
顶点 y 坐标：$y = 2^2 - 4 \times 2 + 3 = 4 - 8 + 3 = -1$
✅ 答案：顶点 $(2, -1)$，对称轴 $x = 2$
✏️ 例题3 进阶

求函数 $y = -x^2 + 2x + 3$ 的最大值。

$a = -1 < 0$，开口朝下，顶点是最高点
对称轴：$x = -\dfrac{2}{2 \times (-1)} = 1$
最大值：$y = -(1)^2 + 2(1) + 3 = -1 + 2 + 3 = 4$
✅ 答案：最大值为 $4$，在 $x = 1$ 时取到
✏️ 例题4 挑战

函数 $y = x^2 - 2x - 3$ 在 $[0, 3]$ 上的最小值和最大值分别是？

第一步：$a=1>0$，开口朝上，顶点是最低点
第二步：对称轴 $x = -\dfrac{-2}{2} = 1$，在 $[0,3]$ 内
第三步：最小值在顶点 $x=1$：$y=1-2-3=-4$
第四步：比较端点值 $f(0)=0-0-3=-3$，$f(3)=9-6-3=0$
第五步：最大值取端点较大值 $f(3)=0$
✅ 答案：最小值 $-4$（$x=1$），最大值 $0$（$x=3$）
🚀 知识点三：指数函数与对数函数
🧬 细菌每小时翻一倍：$1 \to 2 \to 4 \to 8 \to ...$，这就是指数增长 $y = 2^x$。反过来问"翻了几次才到1024？"就是对数：$\log_2 1024 = 10$。指数和对数是一对"互逆操作"。
📐 指数运算法则 $$a^m \cdot a^n = a^{m+n}$$ $$\frac{a^m}{a^n} = a^{m-n}$$ $$(a^m)^n = a^{mn}$$ $$a^0 = 1 \quad (a \neq 0)$$ $$a^{-n} = \frac{1}{a^n}$$
📐 指数函数 $$y = a^x \quad (a > 0, \; a \neq 1)$$ $$a > 1 \text{：递增（越来越大）}$$ $$0 < a < 1 \text{：递减（越来越小）}$$ $$\text{必过定点 } (0, 1)$$
📐 对数 $$a^b = N \iff \log_a N = b$$ $$\text{常用对数：} \lg N = \log_{10} N$$ $$\text{自然对数：} \ln N = \log_e N$$
对数就是"指数的逆运算"。$2^3 = 8$，反过来 $\log_2 8 = 3$（"2的几次方等于8？答：3次"）。

对数运算法则：
• $\log_a (MN) = \log_a M + \log_a N$
• $\log_a \dfrac{M}{N} = \log_a M - \log_a N$
• $\log_a M^n = n \log_a M$
• $\log_a a = 1$，$\log_a 1 = 0$
📐 对数函数 $$y = \log_a x \quad (a > 0, \; a \neq 1, \; x > 0)$$ $$a > 1 \text{：递增}$$ $$0 < a < 1 \text{：递减}$$ $$\text{必过定点 } (1, 0)$$
✏️ 例题1 入门

计算：$2^3 \times 2^2$，$\dfrac{3^5}{3^2}$，$(2^3)^2$

$2^3 \times 2^2 = 2^{3+2} = 2^5 = 32$
$\dfrac{3^5}{3^2} = 3^{5-2} = 3^3 = 27$
$(2^3)^2 = 2^{3 \times 2} = 2^6 = 64$
✅ 答案：$32$，$27$，$64$
✏️ 例题2 入门

求值：$\log_2 8$，$\log_3 27$，$\lg 100$，$\ln e^2$

$\log_2 8 = \log_2 2^3 = 3$
$\log_3 27 = \log_3 3^3 = 3$
$\lg 100 = \lg 10^2 = 2$
$\ln e^2 = 2 \ln e = 2$
✅ 答案：$3, 3, 2, 2$
✏️ 例题3 进阶

比较大小：$2^{0.3}$，$2^{0.5}$，$2^{-1}$

底数 $a = 2 > 1$，指数函数递增
指数越大，函数值越大：$-1 < 0.3 < 0.5$
✅ 答案：$2^{-1} < 2^{0.3} < 2^{0.5}$，即 $\dfrac{1}{2} < 2^{0.3} < \sqrt{2}$
✏️ 例题4 进阶

用对数运算法则化简：$\lg 2 + \lg 5$

$\lg 2 + \lg 5 = \lg(2 \times 5) = \lg 10 = 1$
✅ 答案：$1$
💡 $\lg 2 + \lg 5 = 1$ 是常考结论，建议记住！
✏️ 例题5 挑战

化简：$\lg 25 + \lg 4$

$\lg 25 + \lg 4 = \lg(25 \times 4) = \lg 100 = 2$
✅ 答案：$2$
🧪 第三章 · 小测验
1. 函数 $f(x) = \sqrt{x-2}$ 的定义域是？
💡 根号下 $\geq 0$：$x - 2 \geq 0$，$x \geq 2$。注意是 $\geq$ 不是 $>$，因为 $\sqrt{0} = 0$ 是可以的。
2. 已知 $f(x) = x^2 - 1$，则 $f(3)$ 等于？
💡 $f(3) = 3^2 - 1 = 9 - 1 = 8$。
3. 一次函数 $y = 3x - 2$ 是？
💡 $k = 3 > 0$，所以是递增函数。
4. 二次函数 $y = x^2 - 6x + 5$ 的对称轴是？
💡 $x = -\dfrac{b}{2a} = -\dfrac{-6}{2} = 3$。
5. 函数 $y = -x^2 + 4x$ 的最大值是？
💡 $a=-1<0$，开口朝下，顶点是最高点。$x=-\dfrac{4}{-2}=2$，$y=-(4)+8=4$。
6. $2^{-3}$ 等于？
💡 $2^{-3} = \dfrac{1}{2^3} = \dfrac{1}{8}$。负指数是取倒数，不是变负号！选A的同学把"负指数"和"负号"搞混了。
7. $\log_2 16$ 等于？
💡 $2^4 = 16$，所以 $\log_2 16 = 4$。问自己："2的几次方等于16？"答：4次。
8. $\lg 2 + \lg 50$ 等于？
💡 $\lg 2 + \lg 50 = \lg(2 \times 50) = \lg 100 = 2$。对数相加等于乘积取对数！
9. 指数函数 $y = \left(\dfrac{1}{2}\right)^x$ 是？
💡 底数 $a = \dfrac{1}{2}$，满足 $0 < a < 1$，所以是递减函数。x越大，$\left(\dfrac{1}{2}\right)^x$ 越小。
10. 函数 $f(x) = \dfrac{\sqrt{x}}{\ln x}$ 的定义域不包含下列哪个值？
💡 需要 $x \geq 0$（根号）且 $x > 0$（对数真数）且 $\ln x \neq 0$（分母）。$\ln 1 = 0$，所以 $x=1$ 不在定义域内！
