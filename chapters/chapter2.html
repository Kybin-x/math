第二章 · 不等式

比大小的艺术！学会不等式，就能圈出答案的范围 🎯

不等式的性质 一元一次不等式 一元二次不等式
📋 知识点一：不等式的性质
⚖️ 不等式就像一个天平，左边和右边不一样重。我们对天平两边做同样的操作（加减乘除），天平会怎么变？这就是不等式的性质要告诉你的事情。
📐 不等式的基本性质 $$\text{性质1：} a > b \Rightarrow a+c > b+c$$ $$\text{性质2：} a > b, \; c > 0 \Rightarrow ac > bc$$ $$\text{性质3：} a > b, \; c < 0 \Rightarrow ac < bc$$ $$\text{性质4：} a> b, \; b > c \Rightarrow a > c$$
性质1：两边同时加（或减）同一个数，不等号方向不变。
性质2：两边同时乘以（或除以）一个正数，不等号方向不变。
性质3：两边同时乘以（或除以）一个负数，不等号方向要反转！⚠️
性质4：传递性，大于大于的还是大于。

🔥 最容易出错的就是性质3！乘以负数要翻转不等号，很多同学忘了这一点。
✏️ 例题1 入门

已知 $a > b$，下列哪个一定成立？
A. $2a > 2b$
B. $-a > -b$
C. $a^2 > b^2$
D. $a - b < 0$

A. 两边乘以正数2，不等号不变 → $2a > 2b$ ✅
B. 两边乘以 $-1$（负数），不等号要反转 → $-a < -b$ ❌
C. 不一定，比如 $a=1, b=-2$，$1 < 4$ ❌
D. $a > b$ 意味着 $a-b > 0$ ❌
✅ 答案：A
✏️ 例题2 入门

已知 $a > b > 0$，$c < 0$，比较 $ac$ 和 $bc$ 的大小。

$a > b$，两边乘以负数 $c$
乘以负数，不等号反转
✅ 答案：$ac < bc$
✏️ 例题3 进阶

已知 $a > b$，$c > d$，下列一定成立的是？
A. $a+c > b+d$
B. $ac > bd$
C. $a-c > b-d$
D. $\dfrac{a}{c} > \dfrac{b}{d}$

A. $a > b$ 且 $c > d$，同向不等式可以相加：$a+c > b+d$ ✅
B. 不一定，$a,b,c,d$ 可能有负数，比如 $a=1,b=-1,c=-1,d=-2$，$ac=-1, bd=2$ ❌
C. 不一定，同向不等式不能相减 ❌
D. 不一定，$c,d$ 的正负不确定 ❌
✅ 答案：A
💡 记住：同向不等式可以相加，但不能相减、相乘、相除！
📊 知识点二：一元一次不等式（组）
🚦 解不等式就像找"合格区间"。比如考试60分及格，你的分数 $x$ 要满足 $x \geq 60$。一元一次不等式就是找出 $x$ 的范围。
📐 解一元一次不等式的步骤 $$\text{1. 去分母 → 2. 去括号 → 3. 移项 → 4. 合并同类项 → 5. 系数化为1}$$
和解一元一次方程的步骤完全一样！唯一的区别：
⚠️ 两边乘以或除以负数时，不等号要反转！

不等式组的解法口诀：
"同大取大，同小取小，大小小大取中间，大大小小无解"
即：两个都是 $x >$ 的形式，取较大的那个；两个都是 $x <$ 的形式，取较小的那个。
✏️ 例题1 入门

解不等式：$2x - 3 > 5$

第一步：移项，$2x > 5 + 3 = 8$
第二步：两边除以2（正数，不等号不变），$x > 4$
✅ 答案：$x > 4$，即 $(4, +\infty)$
✏️ 例题2 入门

解不等式：$-3x + 6 \leq 0$

第一步：移项，$-3x \leq -6$
第二步：两边除以 $-3$（负数！不等号反转），$x \geq 2$
✅ 答案：$x \geq 2$，即 $[2, +\infty)$
✏️ 例题3 进阶

解不等式组：$\begin{cases} 2x - 1 > 3 \\ x + 2 < 7 \end{cases}$

第一步：解第一个不等式 $2x - 1 > 3 \Rightarrow 2x > 4 \Rightarrow x > 2$
第二步：解第二个不等式 $x + 2 < 7 \Rightarrow x < 5$
第三步：取交集，$x > 2$ 且 $x < 5$（大小小大取中间）
✅ 答案：$2 < x < 5$，即 $(2, 5)$
✏️ 例题4 进阶

解不等式组：$\begin{cases} 3x + 1 > 10 \\ 2x - 3 > 7 \end{cases}$

第一步：$3x + 1 > 10 \Rightarrow x > 3$
第二步：$2x - 3 > 7 \Rightarrow x > 5$
第三步：两个都是 $x >$，同大取大 → $x > 5$
✅ 答案：$x > 5$，即 $(5, +\infty)$
📈 知识点三：一元二次不等式
🎢 一元二次不等式和一元二次方程是"亲兄弟"。方程是找"等于零的点"，不等式是找"大于零或小于零的区间"。画个抛物线图，一切就清楚了！
📐 一元二次不等式的解法（$a > 0$ 时） $$ax^2 + bx + c = 0 \text{ 的两根为 } x_1 < x_2 \text{ 时：}$$ $$ax^2 + bx + c> 0 \Rightarrow x < x_1 \text{ 或 } x> x_2$$ $$ax^2 + bx + c < 0 \Rightarrow x_1 < x < x_2$$
核心思路：先求方程的根，再看不等式的方向。

当 $a > 0$ 时，抛物线开口朝上（像一个碗 🥣）：
• 大于零（$> 0$）：抛物线在 x 轴上方的部分 → 两根外侧（"大于取两边"）
• 小于零（$< 0$）：抛物线在 x 轴 下方的部分 → 两根中间（"小于取中间"）

💡 口诀："大于取两边，小于取中间"（前提是 $a > 0$）
📐 三种情况汇总（$a > 0$） $$\Delta > 0 \text{（两个不等实根 } x_1 < x_2 \text{）：}$$ $$\quad> 0 \Rightarrow (-\infty, x_1) \cup (x_2, +\infty)$$ $$\quad < 0 \Rightarrow (x_1, x_2)$$ $$\Delta=0 \text{（重根 } x_0 \text{）：}$$ $$\quad> 0 \Rightarrow x \neq x_0 \text{ 的所有实数}$$ $$\quad < 0 \Rightarrow \varnothing \text{（无解）}$$ $$\Delta < 0 \text{（无实根）：}$$ $$\quad> 0 \Rightarrow \mathbb{R} \text{（所有实数）}$$ $$\quad < 0 \Rightarrow \varnothing \text{（无解）}$$
✏️ 例题1 入门

解不等式：$x^2 - 5x + 6 > 0$

第一步：先解方程 $x^2-5x+6=0$，因式分解 $(x-2)(x-3)=0$
第二步：两根 $x_1=2$，$x_2=3$
第三步：$a=1>0$，且是 $>0$，用"大于取两边"
✅ 答案：$x < 2$ 或 $x> 3$，即 $(-\infty, 2) \cup (3, +\infty)$
✏️ 例题2 入门

解不等式：$x^2 - 5x + 6 < 0$

第一步：和上题一样，两根 $x_1=2$，$x_2=3$
第二步：$a=1>0$，且是 $ <0$，用"小于取中间"< /div>
✅ 答案：$2 < x < 3$，即 $(2, 3)$
✏️ 例题3 进阶

解不等式：$2x^2 + 3x - 2 \leq 0$

第一步：解方程 $2x^2+3x-2=0$
第二步：$\Delta=9+16=25$，$x=\dfrac{-3 \pm 5}{4}$
第三步：$x_1=\dfrac{-3-5}{4}=-2$，$x_2=\dfrac{-3+5}{4}=\dfrac{1}{2}$
第四步：$a=2>0$，$\leq 0$ → "小于取中间"（含等号）
✅ 答案：$-2 \leq x \leq \dfrac{1}{2}$，即 $\left[-2, \dfrac{1}{2}\right]$
✏️ 例题4 进阶

解不等式：$-x^2 + 4x - 3 > 0$

第一步：两边乘以 $-1$（不等号反转！）→ $x^2 - 4x + 3 < 0$
第二步：解方程 $x^2-4x+3=0$，$(x-1)(x-3)=0$，$x_1=1, x_2=3$
第三步：现在是 $a=1>0$，$ <0$ → "小于取中间"
✅ 答案：$1 < x < 3$，即 $(1, 3)$
✏️ 例题5 挑战

解不等式：$x^2 - 2x + 1 > 0$

第一步：$x^2-2x+1=(x-1)^2=0$，重根 $x_0=1$
第二步：$\Delta=0$，$(x-1)^2 \geq 0$ 恒成立
第三步：$(x-1)^2 > 0$ 在 $x \neq 1$ 时成立
✅ 答案：$x \neq 1$ 的所有实数，即 $(-\infty,1) \cup (1,+\infty)$
🧪 第二章 · 小测验
1. 已知 $a > b$，下列一定成立的是？
💡 A：乘以 $-1$ 要反转。B：两边加同一个数，不等号不变 ✅。C：$a,b$ 可能有负数。D：$b$ 可能为负。
2. 解不等式 $3x - 6 > 0$ 的解集是？
💡 $3x > 6$，$x > 2$。除以正数3，不等号不变。
3. 解不等式 $-2x \geq 8$ 的解集是？
💡 两边除以 $-2$（负数！），不等号反转：$x \leq -4$。选A的同学忘了反转不等号！
4. 不等式组 $\begin{cases} x > 1 \\ x > 4 \end{cases}$ 的解集是？
💡 同大取大！两个都是 $x >$，取较大的 $x > 4$。
5. 不等式组 $\begin{cases} x > 5 \\ x < 3 \end{cases}$ 的解集是？
💡 大大小小无解！$x$ 不可能同时大于5又小于3。
6. 解不等式 $x^2 - 4x + 3 > 0$ 的解集是？
💡 方程 $x^2-4x+3=0$ 的根为 $x=1, x=3$。$a>0$ 且 $>0$，"大于取两边"：$x <1$ 或 $x> 3$。选A的同学用成了"小于取中间"！
7. 解不等式 $x^2 - 4x + 3 < 0$ 的解集是？
💡 和上题同一个方程，根为1和3。$a>0$ 且 $ <0$，"小于取中间"：$1 < x < 3$。
8. 解不等式 $-x^2 + 2x + 3 > 0$ 的解集是？
💡 两边乘以 $-1$（反转！）→ $x^2-2x-3 < 0$。解方程 $(x+1)(x-3)=0$，根为 $-1$ 和 $3$。$<0$ 取中间：$-1 < x < 3$。
9. 不等式 $(x-1)^2 > 0$ 的解集是？
💡 $(x-1)^2 \geq 0$ 恒成立，等号在 $x=1$ 时取到。所以 $(x-1)^2 > 0$ 的解是 $x \neq 1$。选B的同学忽略了 $x=1$ 时等于0！
10. 不等式 $x^2 + 2x + 5 < 0$ 的解集是？
💡 $\Delta = 4 - 20 = -16 < 0$，方程无实根。$a=1> 0$，抛物线始终在 x 轴上方，即 $x^2+2x+5 > 0$ 恒成立。所以 $ < 0$ 无解。
