第六章 · 平面向量

有大小、有方向——向量就是"带箭头的线段"🏹

基本概念 线性运算 坐标运算 数量积
📌 知识点一：向量的基本概念
🏹 你在导航App 上看到"向北走500 米"——这里既有大小（500米）又有方向（北），这就是一个向量！只有大小、没有方向的量（比如温度、质量）叫标量。
📐 核心概念 $$\text{向量 } \vec{a} \text{ 或 } \overrightarrow{AB} \text{：既有大小又有方向的量}$$ $$|\vec{a}| \text{ 或 } |\overrightarrow{AB}| \text{：向量的模（长度）}$$
几个特殊向量：

🔹 零向量 $\vec{0}$：长度为 $0$，方向不确定
🔹 单位向量：长度为 $1$ 的向量
🔹 相等向量：大小相等、方向相同（与起点无关！）
🔹 相反向量：大小相等、方向相反
🔹 平行向量（共线向量）：方向相同或相反的向量，记为 $\vec{a} \parallel \vec{b}$

⚠️ 注意：零向量与任意向量平行！
➕ 知识点二：向量的线性运算
🚶 你先向东走 3 步，再向北走 4 步，总共位移是多少？把两段路"接起来"——这就是向量的加法！结果是一个从起点到终点的新向量。
📐 向量加法 $$\overrightarrow{AB} + \overrightarrow{BC} = \overrightarrow{AC} \quad \text{（三角形法则：首尾相接）}$$
📐 向量减法 $$\vec{a} - \vec{b} = \vec{a} + (-\vec{b})$$ $$\overrightarrow{OA} - \overrightarrow{OB} = \overrightarrow{BA} \quad \text{（同起点，指向被减数）}$$
📐 数乘向量 $$\lambda \vec{a}\text{：方向与}\vec{a}\text{ 相同}(\lambda > 0)\text{ 或相反}(\lambda < 0)\text{，长度变为 }|\lambda||\vec{a}|$$
💡 运算律（和数的运算一样）：
·交换律：$\vec{a}+\vec{b}=\vec{b}+\vec{a}$
· 结合律：$(\vec{a}+\vec{b})+\vec{c}=\vec{a}+(\vec{b}+\vec{c})$
· 分配律：$\lambda(\vec{a}+\vec{b})=\lambda\vec{a}+\lambda\vec{b}$
📐 共线（平行）条件 $$\vec{a} \parallel \vec{b} \iff \vec{a} = \lambda \vec{b} \quad (\vec{b} \neq \vec{0})$$
✏️ 例题1 入门

化简：$\overrightarrow{AB} + \overrightarrow{BC} + \overrightarrow{CD}$

首尾相接法则：$\overrightarrow{AB}+\overrightarrow{BC}=\overrightarrow{AC}$
$\overrightarrow{AC}+\overrightarrow{CD}=\overrightarrow{AD}$
✅ 答案：$\overrightarrow{AD}$
✏️ 例题2 入门

化简：$\overrightarrow{OA} - \overrightarrow{OB}$

减法口诀：同起点，指向被减数
$\overrightarrow{OA}-\overrightarrow{OB}=\overrightarrow{BA}$
✅ 答案：$\overrightarrow{BA}$
📍 知识点三：向量的坐标运算
📐 建立坐标系后，每个向量都可以用一对数$(x, y)$ 来表示。向量的加减乘除全变成了坐标的加减乘除，计算超方便！
📐 坐标表示 $$\vec{a} = (x_1, y_1), \quad \vec{b} = (x_2, y_2)$$ $$A(x_1,y_1),\ B(x_2,y_2) \implies \overrightarrow{AB} = (x_2-x_1,\ y_2-y_1)$$
📐 坐标运算法则 $$\vec{a} + \vec{b} = (x_1+x_2,\ y_1+y_2)$$ $$\vec{a} - \vec{b} = (x_1-x_2,\ y_1-y_2)$$ $$\lambda\vec{a} = (\lambda x_1,\ \lambda y_1)$$
📐 向量的模 $$|\vec{a}| = \sqrt{x_1^2 + y_1^2}$$ $$|\overrightarrow{AB}| = \sqrt{(x_2-x_1)^2+(y_2-y_1)^2} \quad \text{（两点间距离公式）}$$
📐 平行（共线）的坐标条件 $$\vec{a} \parallel \vec{b} \iff x_1 y_2 - x_2 y_1 = 0$$
💡 记忆口诀：交叉相乘再相减等于零。
$(x_1, y_1)$ 和 $(x_2, y_2)$ → $x_1 y_2 - x_2 y_1 = 0$
✏️ 例题1 入门

已知 $\vec{a}=(3,4)$，求 $|\vec{a}|$。

$|\vec{a}| = \sqrt{3^2+4^2} = \sqrt{9+16} = \sqrt{25} = 5$
✅ 答案：$|\vec{a}| = 5$（经典的3-4-5 直角三角形！）
✏️ 例题2 入门

已知 $A(1,2)$，$B(4,6)$，求 $\overrightarrow{AB}$ 及 $|\overrightarrow{AB}|$。

$\overrightarrow{AB} = (4-1,\ 6-2) = (3,4)$
$|\overrightarrow{AB}| = \sqrt{3^2+4^2} = 5$
✅ 答案：$\overrightarrow{AB}=(3,4)$，$|\overrightarrow{AB}|=5$
✏️ 例题3 进阶

已知 $\vec{a}=(2,3)$，$\vec{b}=(4,k)$，若 $\vec{a} \parallel \vec{b}$，求 $k$。

平行条件：$x_1 y_2 - x_2 y_1 = 0$
$2 \times k - 4 \times 3 = 0$
$2k - 12 = 0$，$k = 6$
✅ 答案：$k = 6$
✏️ 例题4 进阶

已知 $\vec{a}=(1,2)$，$\vec{b}=(3,-1)$，求 $2\vec{a}+\vec{b}$。

$2\vec{a} = (2\times1,\ 2\times2) = (2,4)$
$2\vec{a}+\vec{b} = (2+3,\ 4+(-1)) = (5,3)$
✅ 答案：$2\vec{a}+\vec{b} = (5,3)$
✖️ 知识点四：向量的数量积（内积）
💡 搬一个箱子，力$F$ 和位移 $s$ 都是向量。做的功 $W = Fs\cos\theta$ —— 这就是数量积的物理来源！两个向量"点乘"的结果是一个数（不是向量）。
📐 数量积的定义 $$\vec{a} \cdot \vec{b} = |\vec{a}||\vec{b}|\cos\theta$$$$\text{其中 } \theta \text{ 是 } \vec{a} \text{ 和 } \vec{b} \text{ 的夹角}，0° \leq \theta \leq 180°$$
📐 数量积的坐标公式 $$\vec{a} \cdot \vec{b} = x_1 x_2 + y_1 y_2$$
💡 超级好用！直接把对应坐标相乘再相加就行了。
对比平行条件（交叉相乘相减=0），数量积是对应相乘相加。
📐 重要结论 $$\vec{a} \perp \vec{b} \iff \vec{a} \cdot \vec{b} = 0 \iff x_1 x_2 + y_1 y_2 = 0$$$$\cos\theta = \frac{\vec{a} \cdot \vec{b}}{|\vec{a}||\vec{b}|} = \frac{x_1 x_2 + y_1 y_2}{\sqrt{x_1^2+y_1^2} \cdot \sqrt{x_2^2+y_2^2}}$$ $$|\vec{a}|^2 = \vec{a} \cdot \vec{a} = x_1^2+y_1^2$$
⚠️ 垂直判断三步走：
① 写坐标 →② 对应相乘相加 → ③ 看结果是否为 $0$

⚠️ 数量积不满足结合律：$(\vec{a}\cdot\vec{b})\cdot\vec{c} \neq \vec{a}\cdot(\vec{b}\cdot\vec{c})$
✏️ 例题1 入门

已知 $\vec{a}=(3,4)$，$\vec{b}=(2,-1)$，求 $\vec{a}\cdot\vec{b}$。

$\vec{a}\cdot\vec{b}= 3\times 2 + 4\times(-1) = 6-4 = 2$
✅ 答案：$\vec{a}\cdot\vec{b} = 2$
✏️ 例题2 入门

判断 $\vec{a}=(2,3)$ 和 $\vec{b}=(3,-2)$ 是否垂直。

$\vec{a}\cdot\vec{b} = 2\times3 + 3\times(-2) = 6-6 = 0$
数量积为 $0$，所以 $\vec{a}\perp\vec{b}$
✅ 答案：是，$\vec{a}\perp\vec{b}$
✏️ 例题3 进阶

已知 $\vec{a}=(1,\sqrt{3})$，$\vec{b}=(2,0)$，求 $\vec{a}$ 与 $\vec{b}$ 的夹角 $\theta$。

$\vec{a}\cdot\vec{b} = 1\times2 + \sqrt{3}\times 0 = 2$
$|\vec{a}| = \sqrt{1+3} = 2$，$|\vec{b}| = 2$
$\cos\theta = \dfrac{2}{2\times2} = \dfrac{1}{2}$
$\theta = 60°$
✅ 答案：夹角 $\theta = 60°$
✏️ 例题4 挑战

已知 $\vec{a}=(1,2)$，$\vec{b}=(k,1)$，若 $\vec{a}\perp(2\vec{a}+\vec{b})$，求 $k$。

先算 $2\vec{a}+\vec{b} = (2+k,\ 4+1) = (2+k,\ 5)$
$\vec{a} \perp (2\vec{a}+\vec{b})$ → $\vec{a}\cdot(2\vec{a}+\vec{b})=0$
$1\times(2+k) + 2\times 5 = 0$
$2+k+10 = 0$
$k = -12$
✅ 答案：$k=-12$
🧪 第六章 · 小测验
1. 下列哪个是向量？
💡 向量必须既有大小又有方向。只有B同时具备"大小（10m）+方向（东）"。
2. $\overrightarrow{AB}+\overrightarrow{BA}$ 等于？
💡 $\overrightarrow{BA}=-\overrightarrow{AB}$，所以 $\overrightarrow{AB}+\overrightarrow{BA}=\vec{0}$。（注：$\overrightarrow{AA}$也是$\vec{0}$，但A和C本质一样，选A更标准）
3. 已知 $A(2,1)$，$B(5,5)$，则 $\overrightarrow{AB}$ 等于？
💡 $\overrightarrow{AB} = (5-2,\ 5-1) = (3,4)$。终点减起点！
4. 向量 $\vec{a}=(5,12)$ 的模$|\vec{a}|$ 等于？
💡 $|\vec{a}|=\sqrt{25+144}=\sqrt{169}=13$。经典的 5-12-13 直角三角形。
5. $\vec{a}=(1,2)$，$\vec{b}=(3,4)$，则 $\vec{a}\cdot\vec{b}$ 等于？
💡 $\vec{a}\cdot\vec{b}=1\times3+2\times4=3+8=11$。数量积的结果是一个数，不是向量！
6. $\vec{a}=(4,-3)$，$\vec{b}=(3,4)$，这两个向量的关系是？
💡 $\vec{a}\cdot\vec{b}=4\times3+(-3)\times4=12-12=0$，所以垂直。
7. 已知 $\vec{a}=(2,1)$，$\vec{b}=(1,k)$，若 $\vec{a}\parallel\vec{b}$，则 $k$ 等于？
💡 平行条件：$2\times k - 1\times 1 = 0$，$2k=1$，$k=\dfrac{1}{2}$。
8. $\vec{a}=(2,3)$，则 $3\vec{a}$ 等于？
💡 $3\vec{a}=(3\times2,\ 3\times3)=(6,9)$。数乘就是每个坐标都乘。
9. $\vec{a}=(1,0)$，$\vec{b}=(0,1)$，则 $\vec{a}\cdot\vec{b}$ 等于？
💡 $1\times0+0\times1=0$。$x$ 轴方向和 $y$ 轴方向当然垂直！
10. 已知 $\vec{a}=(1,1)$，$\vec{b}=(1,-1)$，求夹角 $\theta$？
💡 $\vec{a}\cdot\vec{b}=1\times1+1\times(-1)=0$，$\cos\theta=0$，$\theta=90°$。
