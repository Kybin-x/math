🔢 知识点一：乘法公式
🎪 想象你在摆地摊卖东西，有时候需要快速算出两个数的乘积。比如 102 × 98，你会用竖式慢慢算吗？其实有个"秘密武器"——乘法公式，能让你几秒钟就算出来！
📐 平方差公式 $$(a+b)(a-b) = a^2 - b^2$$
两个数的和乘以两个数的差，等于这两个数的平方之差。
简单记忆："和乘差，等于平方差"。
注意：结果是 前面那个数的平方 减去 后面那个数的平方，顺序不能反！
✏️ 例题1 入门

展开：$(x+3)(x-3)$

第一步：识别公式。这里 $a = x$，$b = 3$，符合 $(a+b)(a-b)$ 的形式
第二步：套公式。$a^2 - b^2 = x^2 - 3^2$
第三步：算出结果。$x^2 - 9$
✅ 答案：$(x+3)(x-3) = x^2 - 9$
✏️ 例题2 入门

计算：$102 \times 98$

第一步：把数字拆开。$102 = 100 + 2$，$98 = 100 - 2$
第二步：发现符合平方差公式！$a = 100$，$b = 2$
第三步：套公式。$(100+2)(100-2) = 100^2 - 2^2 = 10000 - 4$
✅ 答案：$102 \times 98 = 9996$
✏️ 例题3 进阶

展开：$(2a+5b)(2a-5b)$

第一步：把 $2a$ 看成一个整体，$5b$ 看成一个整体
第二步：这里"$a$" $= 2a$，"$b$" $= 5b$
第三步：套公式。$(2a)^2 - (5b)^2 = 4a^2 - 25b^2$
✅ 答案：$(2a+5b)(2a-5b) = 4a^2 - 25b^2$
✏️ 例题4 进阶

计算：$53 \times 47$

第一步：$53 = 50 + 3$，$47 = 50 - 3$
第二步：套平方差公式，$a = 50$，$b = 3$
第三步：$50^2 - 3^2 = 2500 - 9 = 2491$
✅ 答案：$53 \times 47 = 2491$
📐 完全平方公式 $$(a+b)^2 = a^2 + 2ab + b^2$$ $$(a-b)^2 = a^2 - 2ab + b^2$$
一个括号的平方，不是简单地把里面每个数平方就行了！
记忆口诀："首平方，尾平方，二倍乘积在中央"
⚠️ 最常见的错误：$(a+b)^2 \neq a^2 + b^2$，千万别忘了中间的 $2ab$！
✏️ 例题5 入门

展开：$(x+4)^2$

第一步：$a = x$，$b = 4$
第二步：首平方 $= x^2$
第三步：尾平方 $= 4^2 = 16$
第四步：二倍乘积 $= 2 \times x \times 4 = 8x$
✅ 答案：$(x+4)^2 = x^2 + 8x + 16$
✏️ 例题6 入门

展开：$(x-5)^2$

第一步：$a = x$，$b = 5$，用的是 $(a-b)^2$ 公式
第二步：首平方 $= x^2$
第三步：尾平方 $= 25$
第四步：二倍乘积 $= 2 \times x \times 5 = 10x$，注意前面是减号
✅ 答案：$(x-5)^2 = x^2 - 10x + 25$
✏️ 例题7 进阶

计算：$101^2$

第一步：$101 = 100 + 1$，所以 $101^2 = (100+1)^2$
第二步：$a = 100$，$b = 1$
第三步：$100^2 + 2 \times 100 \times 1 + 1^2 = 10000 + 200 + 1$
✅ 答案：$101^2 = 10201$
✏️ 例题8 进阶

展开：$(3x+2y)^2$

第一步：$a = 3x$，$b = 2y$
第二步：首平方 $= (3x)^2 = 9x^2$
第三步：尾平方 $= (2y)^2 = 4y^2$
第四步：二倍乘积 $= 2 \times 3x \times 2y = 12xy$
✅ 答案：$(3x+2y)^2 = 9x^2 + 12xy + 4y^2$
➗ 知识点二：分式运算
🍕 分式就像切披萨！一个披萨切成 $b$ 块，你拿了 $a$ 块，你拥有的就是 $\dfrac{a}{b}$。分式运算就是研究怎么把这些"披萨份数"加减乘除。
📐 分式的基本性质 $$\frac{a}{b} = \frac{a \times c}{b \times c} = \frac{a \div c}{b \div c} \quad (c \neq 0)$$
分式的分子和分母同时乘以（或除以）同一个不为零的数，分式的值不变。
就像把一个披萨切成4块拿2块 $\dfrac{2}{4}$，和切成8块拿4块 $\dfrac{4}{8}$，吃到的量是一样的！
📐 分式的四则运算 $$\text{乘法：}\frac{a}{b} \times \frac{c}{d} = \frac{a \times c}{b \times d}$$ $$\text{除法：}\frac{a}{b} \div \frac{c}{d} = \frac{a}{b} \times \frac{d}{c} = \frac{a \times d}{b \times c}$$ $$\text{同分母加减：}\frac{a}{c} \pm \frac{b}{c} = \frac{a \pm b}{c}$$ $$\text{异分母加减：}\frac{a}{b} \pm \frac{c}{d} = \frac{ad \pm bc}{bd}$$
乘法：分子乘分子，分母乘分母，很简单！
除法：除以一个分数 = 乘以它的倒数（把分子分母翻过来）
加减法：分母相同直接加减分子；分母不同要先"通分"（找公分母）
✏️ 例题1 入门

计算：$\dfrac{2}{3} \times \dfrac{5}{7}$

第一步：分子乘分子 $= 2 \times 5 = 10$
第二步：分母乘分母 $= 3 \times 7 = 21$
✅ 答案：$\dfrac{2}{3} \times \dfrac{5}{7} = \dfrac{10}{21}$
✏️ 例题2 入门

计算：$\dfrac{3}{4} \div \dfrac{1}{2}$

第一步：除以一个分数 = 乘以它的倒数
第二步：$\dfrac{3}{4} \times \dfrac{2}{1} = \dfrac{3 \times 2}{4 \times 1} = \dfrac{6}{4}$
第三步：约分 $= \dfrac{3}{2}$
✅ 答案：$\dfrac{3}{4} \div \dfrac{1}{2} = \dfrac{3}{2}$
✏️ 例题3 进阶

计算：$\dfrac{1}{x} + \dfrac{2}{x}$

第一步：分母相同，都是 $x$，直接加分子
第二步：$\dfrac{1+2}{x} = \dfrac{3}{x}$
✅ 答案：$\dfrac{1}{x} + \dfrac{2}{x} = \dfrac{3}{x}$
✏️ 例题4 进阶

计算：$\dfrac{1}{2} + \dfrac{1}{3}$

第一步：分母不同，需要通分。公分母 $= 2 \times 3 = 6$
第二步：$\dfrac{1}{2} = \dfrac{1 \times 3}{2 \times 3} = \dfrac{3}{6}$
第三步：$\dfrac{1}{3} = \dfrac{1 \times 2}{3 \times 2} = \dfrac{2}{6}$
第四步：$\dfrac{3}{6} + \dfrac{2}{6} = \dfrac{5}{6}$
✅ 答案：$\dfrac{1}{2} + \dfrac{1}{3} = \dfrac{5}{6}$
√ 知识点三：二次根式
🌳 什么是"根"？就像一棵树的根一样，$\sqrt{9}$ 就是在问：什么数的平方等于9？答案是3，因为 $3^2 = 9$。这个"找根"的过程就是开平方！
📐 二次根式的性质 $$(\sqrt{a})^2 = a \quad (a \geq 0)$$ $$\sqrt{a^2} = |a|$$ $$\sqrt{ab} = \sqrt{a} \cdot \sqrt{b} \quad (a \geq 0, b \geq 0)$$ $$\sqrt{\frac{a}{b}} = \frac{\sqrt{a}}{\sqrt{b}} \quad (a \geq 0, b > 0)$$
$(\sqrt{a})^2 = a$：先开方再平方，等于没做，回到原来的数。
$\sqrt{a^2} = |a|$：先平方再开方，结果一定是正数（加绝对值）。比如 $\sqrt{(-3)^2} = \sqrt{9} = 3$，不是 $-3$！
乘法：根号里面的数可以分开算。
除法：根号里面的分数可以拆成上下分别开根号。
✏️ 例题1 入门

计算：$\sqrt{16}$

问自己：什么数的平方等于16？
$4 \times 4 = 16$，所以 $\sqrt{16} = 4$
✅ 答案：$\sqrt{16} = 4$
✏️ 例题2 入门

化简：$\sqrt{12}$

第一步：把12拆成一个完全平方数 × 另一个数：$12 = 4 \times 3$
第二步：$\sqrt{12} = \sqrt{4 \times 3} = \sqrt{4} \times \sqrt{3}$
第三步：$\sqrt{4} = 2$
✅ 答案：$\sqrt{12} = 2\sqrt{3}$
✏️ 例题3 进阶

计算：$\sqrt{3} \times \sqrt{27}$

第一步：根号相乘，可以合并。$\sqrt{3} \times \sqrt{27} = \sqrt{3 \times 27}$
第二步：$\sqrt{81} = 9$
✅ 答案：$\sqrt{3} \times \sqrt{27} = 9$
✏️ 例题4 进阶

计算：$3\sqrt{2} + 5\sqrt{2}$

第一步：$\sqrt{2}$ 就像一个"单位"，就像 3个苹果 + 5个苹果 = 8个苹果
第二步：$3\sqrt{2} + 5\sqrt{2} = (3+5)\sqrt{2} = 8\sqrt{2}$
✅ 答案：$3\sqrt{2} + 5\sqrt{2} = 8\sqrt{2}$
🎯 知识点四：一元二次方程
🎯 一元二次方程就是含有一个未知数 $x$，且 $x$ 的最高次数是2的方程。比如 $x^2 - 5x + 6 = 0$。解这种方程就是找出满足条件的 $x$ 的值。
📐 一元二次方程的一般形式 $$ax^2 + bx + c = 0 \quad (a \neq 0)$$
📐 求根公式（万能公式） $$x = \frac{-b \pm \sqrt{b^2 - 4ac}}{2a}$$
📐 判别式 $$\Delta = b^2 - 4ac$$ $$\Delta > 0 \Rightarrow \text{两个不相等的实数根}$$ $$\Delta = 0 \Rightarrow \text{两个相等的实数根（重根）}$$ $$\Delta < 0 \Rightarrow \text{没有实数根}$$
$\Delta$（读作"德尔塔"）就像一个"探测器"，帮你提前判断方程有没有解、有几个解：
$\Delta > 0$：有两个不同的解 ✌️
$\Delta = 0$：有一个解（两个一样的）☝️
$\Delta < 0$：无解 ❌（根号里面是负数，开不了）

解方程的三种常用方法：
① 因式分解法（最快，优先用）
② 求根公式法（万能，什么时候都能用）
③ 配方法（了解即可）
📐 韦达定理（根与系数的关系） $$x_1 + x_2 = -\frac{b}{a}$$ $$x_1 \cdot x_2 = \frac{c}{a}$$
如果方程 $ax^2+bx+c=0$ 有两个根 $x_1$ 和 $x_2$，那么：
两根之和 $= -\dfrac{b}{a}$
两根之积 $= \dfrac{c}{a}$
这个定理很有用！有时候题目不需要你真的把根算出来，只要知道和与积就行。
✏️ 例题1 入门

解方程：$x^2 - 5x + 6 = 0$（因式分解法）

第一步：找两个数，乘积为6，和为5 → 那就是2和3
第二步：因式分解 $x^2 - 5x + 6 = (x-2)(x-3) = 0$
第三步：$x - 2 = 0$ 或 $x - 3 = 0$
✅ 答案：$x_1 = 2$，$x_2 = 3$
✏️ 例题2 入门

解方程：$x^2 - 4 = 0$

第一步：移项 $x^2 = 4$
第二步：两边开方 $x = \pm 2$
✅ 答案：$x_1 = 2$，$x_2 = -2$
✏️ 例题3 进阶

用求根公式解方程：$2x^2 + 3x - 2 = 0$

第一步：确定 $a=2$，$b=3$，$c=-2$
第二步：算判别式 $\Delta = 3^2 - 4 \times 2 \times (-2) = 9 + 16 = 25 > 0$，有两个不等实根
第三步：代入求根公式 $x = \dfrac{-3 \pm \sqrt{25}}{2 \times 2} = \dfrac{-3 \pm 5}{4}$
第四步：$x_1 = \dfrac{-3+5}{4} = \dfrac{2}{4} = \dfrac{1}{2}$
第五步：$x_2 = \dfrac{-3-5}{4} = \dfrac{-8}{4} = -2$
✅ 答案：$x_1 = \dfrac{1}{2}$，$x_2 = -2$
✏️ 例题4 进阶

已知方程 $x^2 - 3x + 1 = 0$ 的两根为 $x_1, x_2$，求 $x_1^2 + x_2^2$ 的值。

第一步：由韦达定理，$x_1 + x_2 = 3$，$x_1 \cdot x_2 = 1$
第二步：利用公式 $x_1^2 + x_2^2 = (x_1+x_2)^2 - 2x_1 x_2$
第三步：$= 3^2 - 2 \times 1 = 9 - 2 = 7$
✅ 答案：$x_1^2 + x_2^2 = 7$
📏 知识点五：平面图形面积
🏠 装修房子要算地板面积、种草坪要算草地面积……面积公式是生活中最实用的数学知识之一！这里复习几个最常用的图形面积公式。
📐 常用面积公式 $$S_{\text{三角形}} = \frac{1}{2} \times \text{底} \times \text{高}$$ $$S_{\text{矩形}} = \text{长} \times \text{宽}$$ $$S_{\text{平行四边形}} = \text{底} \times \text{高}$$ $$S_{\text{梯形}} = \frac{1}{2}(\text{上底} + \text{下底}) \times \text{高}$$ $$S_{\text{圆}} = \pi r^2$$
三角形：底 × 高 ÷ 2，记住一定要除以2！
平行四边形：和矩形类似，但注意"高"是垂直距离，不是斜边！
梯形：上底加下底的和 × 高 ÷ 2，口诀"上下底加起来乘高除以二"
圆：$\pi r^2$，$r$ 是半径。$\pi \approx 3.14$
✏️ 例题1 入门

一个三角形底边长8cm，高5cm，求面积。

公式：$S = \dfrac{1}{2} \times \text{底} \times \text{高}$
$S = \dfrac{1}{2} \times 8 \times 5 = 20 \text{ cm}^2$
✅ 答案：面积为 $20 \text{ cm}^2$
✏️ 例题2 入门

一个圆的半径为3cm，求面积。

公式：$S = \pi r^2$
$S = \pi \times 3^2 = 9\pi \approx 28.26 \text{ cm}^2$
✅ 答案：面积为 $9\pi \text{ cm}^2$（约 $28.26 \text{ cm}^2$）
✏️ 例题3 进阶

一个梯形上底4cm，下底10cm，高6cm，求面积。

公式：$S = \dfrac{1}{2}(\text{上底} + \text{下底}) \times \text{高}$
$S = \dfrac{1}{2} \times (4 + 10) \times 6 = \dfrac{1}{2} \times 14 \times 6 = 42 \text{ cm}^2$
✅ 答案：面积为 $42 \text{ cm}^2$
✏️ 例题4 进阶

一个扇形的半径为6cm，圆心角为60°，求扇形面积。

扇形面积公式：$S = \dfrac{n}{360} \times \pi r^2$，其中 $n$ 是圆心角度数
$S = \dfrac{60}{360} \times \pi \times 6^2 = \dfrac{1}{6} \times 36\pi = 6\pi \approx 18.84 \text{ cm}^2$
✅ 答案：面积为 $6\pi \text{ cm}^2$（约 $18.84 \text{ cm}^2$）
📐 知识点六：勾股定理
🏗️ 古代工匠建房子的时候，怎么确认墙角是直角？他们用一根绳子，量出3、4、5的比例，就能得到一个完美的直角！这就是勾股定理的实际应用。
📐 勾股定理 $$a^2 + b^2 = c^2$$
在一个直角三角形中，两条直角边（$a$ 和 $b$）的平方和，等于斜边（$c$）的平方。
简单记忆："两条短边的平方加起来 = 最长边的平方"

常见的勾股数组（建议背下来）：
• $3, 4, 5$（以及它的倍数 $6,8,10$ 等）
• $5, 12, 13$
• $8, 15, 17$
✏️ 例题1 入门

直角三角形两直角边分别为3和4，求斜边长。

$c^2 = a^2 + b^2 = 3^2 + 4^2 = 9 + 16 = 25$
$c = \sqrt{25} = 5$
✅ 答案：斜边长为 $5$
✏️ 例题2 入门

直角三角形斜边为13，一条直角边为5，求另一条直角边。

$a^2 + 5^2 = 13^2$
$a^2 = 169 - 25 = 144$
$a = \sqrt{144} = 12$
✅ 答案：另一条直角边为 $12$
✏️ 例题3 进阶

判断三边为 $7, 24, 25$ 的三角形是否为直角三角形。

第一步：最长边为25，验证 $7^2 + 24^2$ 是否等于 $25^2$
第二步：$7^2 + 24^2 = 49 + 576 = 625$
第三步：$25^2 = 625$
第四步：$625 = 625$ ✓
✅ 答案：是直角三角形
✏️ 例题4 进阶

一个长方形的长为8cm，对角线为10cm，求宽。

第一步：长方形的对角线把长方形分成两个直角三角形
第二步：长和宽是直角边，对角线是斜边
第三步：$8^2 + \text{宽}^2 = 10^2$
第四步：$\text{宽}^2 = 100 - 64 = 36$
第五步：$\text{宽} = 6$ cm
✅ 答案：宽为 $6$ cm
🧪 第零章 · 小测验
1. 展开 $(x+6)(x-6)$ 的结果是？
💡 这是平方差公式 $(a+b)(a-b)=a^2-b^2$，所以 $(x+6)(x-6)=x^2-36$。
2. $(x+3)^2$ 展开后等于？
💡 完全平方公式 $(a+b)^2=a^2+2ab+b^2$，这里 $a=x, b=3$，所以 $=x^2+6x+9$。最常见的错误就是选A，忘了中间的 $2ab$ 项！
3. $\dfrac{2}{5} \div \dfrac{4}{3}$ 的结果是？
💡 除以一个分数等于乘以它的倒数：$\dfrac{2}{5} \times \dfrac{3}{4} = \dfrac{6}{20} = \dfrac{3}{10}$。注意C选项没有约分！
4. $\dfrac{1}{3} + \dfrac{1}{6}$ 的结果是？
💡 通分：公分母为6，$\dfrac{1}{3}=\dfrac{2}{6}$，所以 $\dfrac{2}{6}+\dfrac{1}{6}=\dfrac{3}{6}=\dfrac{1}{2}$。
5. $\sqrt{50}$ 化简后等于？
💡 $50=25 \times 2$，所以 $\sqrt{50}=\sqrt{25}\times\sqrt{2}=5\sqrt{2}$。
6. $2\sqrt{3} + 4\sqrt{3}$ 的结果是？
💡 同类根式相加，系数相加即可：$(2+4)\sqrt{3}=6\sqrt{3}$。根号里面的数不变！
7. 方程 $x^2 - 7x + 12 = 0$ 的两个根是？
💡 找两个数，乘积为12，和为7 → 3和4。因式分解：$(x-3)(x-4)=0$，所以 $x=3$ 或 $x=4$。
8. 方程 $x^2 + 2x + 5 = 0$ 的判别式 $\Delta$ 等于多少？该方程有实数根吗？
💡 $\Delta=b^2-4ac=4-20=-16 < 0$，所以没有实数根。根号里面是负数，开不了！
9. 一个梯形上底3cm，下底7cm，高4cm，面积是？
💡 $S=\dfrac{1}{2}(3+7)\times 4=\dfrac{1}{2}\times 10\times 4=20 \text{ cm}^2$。选B的同学忘了除以2！
10. 一个圆的直径为10cm，面积是？
A. $10\pi \text{ cm}^2$
💡 直径10cm，所以半径 $r=5$ cm。$S=\pi r^2=\pi \times 25=25\pi \text{ cm}^2$。选C的同学把直径当半径了！
11. 直角三角形两直角边分别为6和8，斜边长为？
💡 $c^2=6^2+8^2=36+64=100$，$c=\sqrt{100}=10$。这就是经典的 $3,4,5$ 的2倍：$6,8,10$。
12. 三边长分别为 $5, 12, 13$ 的三角形是直角三角形吗？
💡 验证：$5^2+12^2=25+144=169=13^2$ ✓ 满足勾股定理，所以是直角三角形。$5,12,13$ 是常见勾股数组，建议记住！
